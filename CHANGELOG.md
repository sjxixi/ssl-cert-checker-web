## [1.0.0] - 2025-12-16

### 新增
- ✨ 单个域名SSL证书查询功能
- ✨ 批量域名查询（支持并发）
- ✨ SAN域名列表展示
- ✨ 关注域名管理（添加、删除、备注）
- ✨ 手动录入证书信息
- ✨ 批量操作（刷新、导出、删除）
- ✨ 批量导入（CSV/TXT格式）
- ✨ 通知预警功能（可配置阈值）
- ✨ 历史记录功能
- ✨ 数据可视化图表
  - 证书状态分布饼图
  - 剩余天数分布柱状图
  - 证书颁发者分布
  - 即将过期域名列表
- ✨ 高级过滤搜索
- ✨ CSV导出功能
- ✨ 系统设置页面
- ✨ 浅色/深色主题切换
- ✨ Logo点击返回首页

### 优化
- 🎨 界面采用淡雅白色主题，视觉舒适
- 🎨 深色主题完整适配
- 🎨 响应式布局，适配不同屏幕
- 🎨 证书状态四级分类（安全、警告、危险、过期）
- ⚡ 并发查询提升批量查询性能
- ⚡ SQLite数据库本地持久化
- ⚡ 前后端分离架构

### 修复
- 🐛 修复饼图数量为0时仍显示的问题
- 🐛 修复证书颁发者柱状图未对齐的问题
- 🐛 修复深色主题未生效的问题
- 🐛 修复GetWatchedDomains未定义错误

### 技术栈
- **后端**: Go 1.21 + Wails v2.11.0
- **前端**: 原生 JavaScript ES6+
- **数据库**: SQLite
- **构建**: Vite

## 版本说明

### 版本号格式: MAJOR.MINOR.PATCH

- **MAJOR**: 不兼容的API修改
- **MINOR**: 向下兼容的功能新增
- **PATCH**: 向下兼容的问题修复

### 更新类型标识

- ✨ `新增` - 新功能
- 🎨 `优化` - 改进和优化
- 🐛 `修复` - Bug修复
- 📝 `文档` - 文档更新
- ⚡ `性能` - 性能优化
- 🔒 `安全` - 安全修复
- ♻️ `重构` - 代码重构
- 🗑️ `移除` - 功能移除
- 💥 `破坏性变更` - 不兼容的变更
